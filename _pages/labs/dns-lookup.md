## DNS Lookup Lab

### Objective
Perform passive reconnaissance using DNS query tools to gather domain information.

### Tools Used
- nslookup
- dig
- whois

### Steps
1. Queried DNS A, MX, NS records
2. Identified subdomains
3. Retrieved registrar information

### Key Learnings
- DNS can reveal server locations
- Subdomain enumeration improves mapping
- Passive recon avoids triggering IDS
